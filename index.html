Welcome to the home page for the SCB study of Recovery Plans for threatened and endangered species. Here you can find information about the project and links to the database.

Website Contents:

Project Synopsis. The Synopsis page gives a brief description of the Recovery Plan study, including its goals and general methods.

Publications. The Publications page presents a bibliography of published papers that developed from the Recovery Plan project.

Project Participants. The Participants page provides a comlete list of the 325+ researchers from 19 universities who partipated in this project.

Data Collection. The Data Collection page provides links to the data-collection Instrument - the original questionnaire used to gather data from each Recovery Plan reviewed - and an analysis of how consistently data were collected by project participants.

Data. The Data page provides links to a list of the Recovery Plans reviewed during this study, as well as the raw data for each plan.
